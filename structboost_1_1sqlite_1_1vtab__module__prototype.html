Detailed Description

The prototype used by create_module.

Note that the class names are for exposition only - the types are deduced by the library.

Please see the sqlite reference for the virtual table documentation vtab.

Error handling is implemented through exceptions. Members are required unless explicitly stated optional. The destruction functions are automatically filled with destructors.
